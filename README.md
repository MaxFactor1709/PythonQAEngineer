# PythonQAEngineer
Репозиторий для домашних работ
Что сейчас на проектах?
У нас в компании 3 проекта. Тестирование САФО, тестирование Ефакторинга, и тестирование ВРМ и CRM.
Как организовано тестирование? 
У нас есть TFS, в котором описываются user story, которые описывают изменения в системах.
По результатам проверки этих задач мы организуем пользовательское тестирование критичного функционала. По результатам 
которого принимается решение о внедрении  нового функционала. Параллельно мы проводим регрессионное тестирование 
функционала и обновляем кейсы для проведения регрессионного тестирования.
Чего хочу добиться я? Сокращение времени на регрессионное тестирование с помощью автоматизации. Реорганизовать отдел
тестирования. Внедрить автоматизацию тестирования на проектах. Это глобально.
